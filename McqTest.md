###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: ?

<i>Output: "12"</i>
<br/>
<i>JavaScript is a dynamically typed language, and it will attempt to perform type coercion to convert one or both operands to a common type before performing the addition. In this case, it will concatenate the string "2" to the number 1, resulting in the string "12"</i>

</p>
</details>
